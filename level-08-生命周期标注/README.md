# Level 08：生命周期标注

## 通关标准

> 本关结束后，你能手写生命周期标注语法，能理解生命周期省略规则（Elision），能向 C/C++ 开发者用"编译期悬垂指针检查器"的视角解释 Rust 生命周期的本质。

## 核心概念速查

| Rust 概念 | 对比 C/C++ | 对比 GC 语言 |
|-----------|-----------|-------------|
| 生命周期参数 `'a` | 无直接对应 | 无（GC 管理） |
| Elision 规则 | 无 | 无 |
| `'static` | static 存储期 | N/A |
| 结构体中的引用字段 | 裸指针（危险） | 自然支持 |
| 函数返回引用 | 悬垂指针（UB） | GC 保证安全 |

## 生命周期标注速查

```rust
&'a T          // 引用，生命周期 'a
&'a mut T      // 可变引用，生命周期 'a
fn f<'a>(x: &'a T) -> &'a T  // 输入输出同一生命周期
fn f<'a, 'b>(x: &'a T, y: &'b U) -> &'a T  // 多个生命周期
```

## 生命周期省略规则（Elision）

编译器会自动推断以下模式的生命周期，无需手动标注：

1. 每个引用参数获得自己的生命周期参数
2. 如果只有一个输入生命周期，输出生命周期等于它
3. 如果方法有 `&self`/`&mut self`，输出生命周期等于 `self` 的生命周期

