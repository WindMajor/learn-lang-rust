# Level 00：前置学习 —— Rust 基础语法速览

> ⚠️ 本关是"参考手册"而非"闯关关卡"。你已经掌握 8 门语言，这里的 let/mut/if/for 只是帮你快速对上 Rust 的"方言"。
> 如果你对 Rust 基础语法已经很熟，可以**直接跳到 Level 01**。

## 通关标准

> 本关结束后，你不需要"学完"，只需要确认：**看一眼 Rust 语法就知道它对应你已知语言的哪个写法**。放轻松，快速浏览即可。

## 13 个速览模块

| 模块 | 文件 | 内容 | 跳过建议 |
|------|------|------|----------|
| 01 | `variables` | 不可变/可变变量、常量、遮蔽 | 选看 |
| 02 | `data_types` | 标量/复合类型、`&str` vs `String` | 选看 |
| 03 | `functions` | 函数、闭包、函数指针、方法 | 选看 |
| 04 | `control_flow` | if/match/loop/while/for/let else | **快速浏览** |
| 05 | `ownership` | 所有权规则、引用/借用、切片 | **必看**（核心分水岭） |
| 06 | `structs` | 结构体、方法、derive | **必看** |
| 07 | `enum_and_pattern_matching` | 枚举、Option/Result、模式匹配 | **必看** |
| 08 | `collections` | Vec/HashMap/HashSet/VecDeque/... | 选看 |
| 09 | `modules` | 模块系统、pub、use | **必看** |
| 10 | `error_handling` | panic/Result/? /自定义错误 | **必看** |
| 11 | `generics` | 泛型函数/结构体、单态化 | 选看 |
| 12 | `traits` | trait 定义/实现/约束/trait 对象 | **必看** |
| 13 | `lifetimes` | 生命周期标注/省略规则/Pin/Cow | **必看** |

```

05: 所有权预览          →      L02+L03 深入所有权+借用
06: 结构体预览          →      L05 结构体+枚举+模式匹配
07: 枚举/模式匹配       →      L05 同上
08: 集合操作            →      L04 字符串+集合+L09 迭代器

13: 生命周期            →      L08 核心学习
─────────────────────        ─────────────────────────
```

## 编译/运行命令

```bash
cargo run          # 一次性运行 13 个模块的演示代码
```
